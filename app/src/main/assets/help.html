Help File

Unique feature like visual Pill Reminders (Medicine image will flash which will remind you its time to take medicine), SMS scheduler (a reminder SMS will be sent automatically to a pre defined mobile number), Multiple Patient profiles (along with the patient picture), multiple medications per profile • You are able to add multiple dosage times, and which days of the week they apply to, which in turn set the application to sound a visual/audio reminder.

Add Patient

Add patient button will open a screen where user can enter information for the medication you want to set reminder for.

Patient Pic

Use the camera button to take the picture for the individual patient profile. You can create multiple patient profiles identified by different pictures.

Ring Tone

Select the desired ringtone to go off when the reminder is triggered.

No. Of Pills

Set number of pills for the reminder. You can add the dosages as per your prescription

Potency

Add the potency of the dose

Add Record

Use the plus button to add the medication information along with the alarm/reminder into the data base.

Course Duration

Set the duration of the medication course ranging from days to months to year to life long options.

Medication Time

Set the time(s) for the day when the pill reminder need to be set. You can chose multiple times in a day for the number of pill you need to take.

RecurrenceTime

Choose from range of options for reminder recurrence

 

Auto SMS Feature

Go to Settings and use the auto SMS option to send a reminder SMS when it is time to take medicine. You can use "Add Mobile Number" option to add the mobile no. for the same
